## API Stack

### GraphQL
- **Endpoint**: `/api/graphql`
- **Schema**: Type-safe GraphQL API
- **Documentation**: See autogeneration docs above

### REST
- **Endpoint**: `/api/rest/*`
- **Documentation**: OpenAPI/Swagger spec (autogenerated)

### tRPC
- **Endpoint**: `/api/trpc/*`
- **Type Safety**: End-to-end TypeScript
- **Documentation**: See server implementation in `src/server/trpc/`

## API Development

### Adding New Endpoints

1. Define schema in appropriate module
2. Implement resolver/handler
3. Run autogeneration: `pnpm generate:api`
4. Update API documentation
5. Add tests

### API Testing

```bash
# GraphQL Playground
http://localhost:3000/api/graphql

# tRPC Playground
http://localhost:3000/api/trpc-playground
```
