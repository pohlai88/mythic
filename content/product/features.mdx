# Product Features

## Core Features

### Document Management

- Sealed document support with hash verification
- Amendment tracking and history
- Version control integration

### Governance Components

- Status badges for document status
- Constitutional citations
- Reference benchmark displays
- Amendment history visualization

### Navigation

- Layer-based organization (L0/L1/L2)
- Auto-generated navigation from file structure
- Cross-reference links

## Technical Features

### Built with Next.js

- Server-side rendering
- Static site generation
- API routes support

### Nextra Documentation

- MDX support
- Built-in search
- Dark mode
- Mobile responsive

### Verification

- SHA-256 hash computation
- Ledger entry integration
- Integrity verification
