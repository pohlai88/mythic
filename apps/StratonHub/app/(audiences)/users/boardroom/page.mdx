# BoardRoom User Guide

> Complete guide for using the BoardRoom executive decision engine

## Overview

BoardRoom is the executive decision engine that enables fast, accountable decision-making for your organization.

**Purpose**: Transform executive governance from "Slow Email" to "Fast Boardroom" with zero-latency decisions and forensic accountability.

**Key Features**:
- Proposal creation and management
- Real-time approval workflows
- Audit trail tracking
- Decision impact analysis

## Quick Start

Get started with BoardRoom in 3 steps:

1. **Create a Proposal**: Navigate to BoardRoom and click "New Proposal"
2. **Fill in Details**: Use the stencil form to provide proposal information
3. **Submit for Approval**: Submit to the appropriate circle for review

## Getting Started

### Creating Your First Proposal

1. Navigate to the BoardRoom module
2. Click the "New Proposal" button
3. Select a stencil (e.g., "Hiring Request", "Budget Expansion")
4. Fill in the required fields
5. Submit for approval

### Understanding Proposal Status

- **DRAFT**: Proposal is being created
- **SUBMITTED**: Proposal is awaiting approval
- **APPROVED**: Proposal has been approved
- **VETOED**: Proposal has been rejected

## Common Tasks

### Submitting a Proposal

1. Create a new proposal
2. Fill in all required fields
3. Review the proposal details
4. Click "Submit for Approval"
5. Select the appropriate circle
6. Confirm submission

### Tracking Proposal Status

1. Navigate to "My Proposals"
2. View the status of each proposal
3. Click on a proposal to see details
4. Review the approval history

## Troubleshooting

### Issue: Cannot submit proposal

**Symptoms**: Submit button is disabled or shows an error

**Cause**: Required fields are missing or invalid

**Solution**: Review the form and ensure all required fields are filled correctly

### Issue: Proposal not appearing in list

**Symptoms**: Created proposal doesn't show in proposals list

**Cause**: Browser cache or filter settings

**Solution**: Refresh the page or check filter settings
