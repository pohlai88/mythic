# intelligentVarianceStyles

Intelligent variance styling

Automatically styles variance displays based on percentage thresholds.

## Parameters

### variancePct

**Type**: `number`

**Required**

- Variance percentage


### variant

**Type**: `"badge" | "text" | "indicator"`

**Optional**

- Display variant


### className

**Type**: `string | undefined`

**Optional**

- Additional classes to merge



## Returns

`string`
