# intelligentResponsiveStyles

Intelligent responsive breakpoint utilities

Automatically applies responsive classes based on content type.

## Parameters

### type

**Type**: `"card" | "form" | "dashboard" | "list"`

**Required**

- Content type (dashboard, card, list, form)


### className

**Type**: `string | undefined`

**Optional**

- Additional classes to merge



## Returns

`string`
