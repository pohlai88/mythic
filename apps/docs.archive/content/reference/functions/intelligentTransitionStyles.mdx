# intelligentTransitionStyles

Intelligent transition utilities

Applies Axis Visual Canon compliant transitions based on interaction type.

## Parameters

### type

**Type**: `"hover" | "commit" | "illuminate"`

**Optional**

- Interaction type (hover, commit, illuminate)


### className

**Type**: `string | undefined`

**Optional**

- Additional classes to merge



## Returns

`string`
