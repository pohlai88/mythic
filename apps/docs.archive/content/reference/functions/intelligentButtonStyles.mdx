# intelligentButtonStyles

Intelligent button styling utility

Provides consistent button styling across the application with intelligence-driven variants.

## Parameters

### variant

**Type**: `"primary" | "secondary" | "danger"`

**Optional**

- Button variant (primary, secondary, danger)


### size

**Type**: `"sm" | "md" | "lg"`

**Optional**

- Button size (sm, md, lg)


### className

**Type**: `string | undefined`

**Optional**

- Additional classes to merge



## Returns

`string`
