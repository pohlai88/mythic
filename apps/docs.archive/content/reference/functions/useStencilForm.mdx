Hook for managing stencil-based forms with React Hook Form + Zod

**Type**: `T`

**Required**

No description available



## Returns

```typescript
```tsx

<form onSubmit={handleSubmit(onSubmit)}>
  <input {...register('job_title')} />

</form>
