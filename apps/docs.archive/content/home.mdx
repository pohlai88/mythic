# Welcome to NexusCanon Documentation

This is a custom documentation site built with Next.js App Router and Tailwind CSS v4.

## Features

- **100% Tailwind CSS**: All UI uses design system tokens (void, parchment, gold, ash, charcoal)
- **Custom Content Automation**: Automatic page discovery and navigation
- **MDX Support**: Rich content with custom components
- **Design System Integration**: Full compliance with Axis Visual Canon
- **Responsive Design**: Mobile-first with Tailwind breakpoints
- **Dark Mode**: Theme switching support

## Getting Started

Explore the documentation using the navigation menu. All content is automatically discovered and organized.

## Architecture

This documentation system uses:
- **Next.js App Router** for routing and rendering
- **Tailwind CSS v4** for styling (CSS-first configuration)
- **Custom MDX Processing** with `next-mdx-remote`
- **Design System Tokens** from `@mythic/design-system`
- **Intelligence-Driven Utilities** for contextual styling
