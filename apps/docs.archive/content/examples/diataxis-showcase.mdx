## Tailwind Intelligence in Action

### Intelligence-Driven Styling

All Diátaxis components now use intelligence-driven utilities from `@mythic/shared-utils`:

```tsx
// DocumentTypeBadge uses intelligentStatusStyles()
<DocumentTypeBadge type="tutorial" />
// Automatically applies APPROVED status styling (success colors)

// DocumentTypeBanner uses multiple intelligence utilities
<DocumentTypeBanner type="how-to" />
// Uses: intelligentStatusStyles + intelligentGradientStyles + intelligentTransitionStyles

// HowToGuide uses intelligence for problem/solution sections
<HowToGuide 
  problem="..." 
  solution="..."
  steps={[...]}
/>
// Problem uses VETOED status (error colors)
// Solution uses APPROVED status (success colors)
// Steps use LISTENING status (neutral colors)
```

### Using Intelligence Functions Directly

```tsx
import { 
  getDiataxisIntelligenceStyles,
  getDiataxisGradientStyles,
  getDiataxisCombinedStyles 
} from '@/lib/diataxis-intelligence'

// Get intelligence styles for document type
const badgeStyles = getDiataxisIntelligenceStyles('tutorial', 'badge')
const borderStyles = getDiataxisIntelligenceStyles('how-to', 'border')

// Get combined styles with gradient and transition
const combinedStyles = getDiataxisCombinedStyles('tutorial', {
  includeGradient: true,
  includeTransition: true,
  transitionType: 'illuminate'
})
```
