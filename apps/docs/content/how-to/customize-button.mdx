# How to Customize a Button

This guide shows you how to customize button styles and variants.

## Problem

You need a button with custom styling that matches your design requirements.

## Solution

### Step 1: Use Existing Variants

The Button component supports multiple variants:

```typescript
<Button variant="primary">Primary</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="outline">Outline</Button>
```

### Step 2: Add Custom Classes

You can extend button styles with Tailwind classes:

```typescript
<Button className="bg-custom-color text-white">
  Custom Button
</Button>
```

### Step 3: Create a New Variant

For reusable custom variants, extend the Button component:

```typescript
// components/CustomButton.tsx
import { Button } from '@mythic/design-system/components'
import { cva } from 'class-variance-authority'

const customButtonVariants = cva(
  'base-classes',
  {
    variants: {
      variant: {
        custom: 'custom-styles',
      },
    },
  }
)

export function CustomButton({ variant, ...props }) {
  return <Button className={customButtonVariants({ variant })} {...props} />
}
```
