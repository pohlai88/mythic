# @mythic/shared-utils

Shared utility functions for the ERP monorepo.

## Usage

### Class Name Utility (cn)

```typescript
import { cn } from "@mythic/shared-utils"

// Combine classes with Tailwind merging
const className = cn("px-2 py-1", "bg-red-500", isActive && "bg-blue-500")
```

## Exports

- `cn()` - Class name utility with Tailwind CSS class merging
