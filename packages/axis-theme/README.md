# @mythic/axis-theme

Axis Visual Canon - Complete Theme System

BEASTMODE: Supreme Luxury Gold (#c9a961) + Comfortable Reading White (Parchment #f8f6f0)

## Usage

```typescript
import { axisColors, axisTypography, axisMotion } from '@mythic/axis-theme'

// Colors
const gold = axisColors.gold // #c9a961 - Supreme luxury gold

// Typography
const editorialFont = getEditorialFont() // Cormorant Garamond
const forensicFont = getForensicFont() // JetBrains Mono

// Motion
const hoverTransition = getHoverTransition('background-color') // 1000ms gravitational
const commitmentTransition = getCommitmentTransition('transform') // 1618ms golden ratio
```

## Features

- **BEASTMODE Gold Palette** - Supreme luxury colors
- **Material Truth Design** - Colors represent material states
- **Typography Doctrine** - Editorial + Forensic fonts
- **Gravitational Motion** - No bounce, snap, or elastic easing
